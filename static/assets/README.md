## Image Guidelines

### Quality
- Use high-resolution images (at least 1920px wide for hero images)
- Optimize for web (use JPG for photos, PNG for logos)
- Aim for < 500KB per image after optimization

### Style
- Professional, high-quality travel photography
- Consistent color grading across images
- Avoid overly stock-looking images
- Show real destinations and experiences

### Sources
You can get free high-quality images from:
- Unsplash.com
- Pexels.com
- Pixabay.com
- Or use your own professional photography

### Optimization Tools
- TinyPNG.com - Compress images
- Squoosh.app - Google's image optimizer
- ImageOptim (Mac) - Desktop image optimizer

## Important Notes

1. **CEO Photo** - This is critical! Make sure to add a professional photo of the CEO
2. All file names should match exactly (case-sensitive)
3. Hero images work best as landscape orientation
4. Team photos work best as square or portrait

Once you add all images, the website will look complete and professional!
